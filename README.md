This integrative project aims to apply the knowledge acquired throughout the technical course integrated with high school, developed by a group of five students, focusing on solving a practical problem through an interdisciplinary approach.

## Accessing the Project

You can access the project's source code or download it directly from this repository.

## Running the Project Locally

After downloading the project into your XAMPP `htdocs` folder, follow these steps to run it in your browser:

1.  Open the **XAMPP Control Panel** and start the **Apache** server.
2.  Open your web browser.
3.  Type `localhost/` followed by the name of the project folder in the address bar.
    - *Example: `localhost/pipa-folder`*
