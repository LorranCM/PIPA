#Description
This integrative project aims to apply the knowledge acquired throughout the technical 
course integrated with high school, developed by a group of five students, focusing on 
solving a practical problem through an interdisciplinary approach. 

#Acesso ao Projeto
Você pode acessar o código fonte do projeto ou baixá-lo neste repositório.

#Abrir e Rodar o Projeto
Após baixar o projeto dentro da sua pasta htdocs do Xampp, você pode abrir no seu navegador:
- Abra o Xampp e inicie o servidor Apache;
- Abra seu navegador e digite "localhost/" e o nome da pasta que você salvou o projeto depois da barra
